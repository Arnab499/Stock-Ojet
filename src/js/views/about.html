About Stock Manager

  1. Key Features:
    • Stock Management: Add new stocks with name, price, and quantity; remove stocks by unique ID; update existing stock records (price or quantity); retrieve stock details from the system.
    • Customer Management: Register new customers with name, customer ID, and email; remove customers by ID; update existing customer records; retrieve customer details from the system.
    • Stock Transactions: Purchase stocks — decreases inventory quantity and logs the transaction in the customer’s history; Sell stocks — increases inventory quantity and records the sale in the customer’s history.
    • Data Persistence: Oracle database integration for storing stock, customer, and transaction data persistently, ensuring data integrity across sessions.
  2. Technologies Used:
    • Programming Language: Java version 17
    • Framework: Spring Boot
    • Server: Apache Tomcat
    • Database: Oracle Database version 19c
